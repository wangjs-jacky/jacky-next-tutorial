# Next.js 入门指南

Next.js 是一个强大的 React 框架，提供了许多开箱即用的功能。

## 主要特性

- **服务端渲染**：提高性能和 SEO
- **静态生成**：预渲染页面
- **API 路由**：构建全栈应用
- **文件系统路由**：自动路由生成

## 开始使用

使用 create-next-app 快速创建项目：

```bash
npx create-next-app@latest
```
