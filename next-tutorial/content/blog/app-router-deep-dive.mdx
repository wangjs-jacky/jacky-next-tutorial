# App Router 深入解析

App Router 是 Next.js 13+ 引入的新路由系统。

## 核心概念

- **布局**：共享 UI 组件
- **页面**：路由段
- **加载状态**：显示加载 UI
- **错误处理**：捕获错误

## 优势

App Router 提供了更好的开发体验和性能优化。

I **love** using [Next.js](https://nextjs.org/)
